# Taskflow

A task management web app built with HTML, CSS, JavaScript, and PHP.

## Features
- User authentication (login/logout)
- Dashboard with task overview
- Project & task management
- Kanban board view
- Timeline view
- Team management
- Reports & activity log
- Notifications

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

## Setup Instructions
1. Install [XAMPP](https://www.apachefriends.org/)
2. Clone the repo into your `htdocs` folder
3. Import `database/taskflow.sql` into phpMyAdmin
4. Copy `api/config.example.php` to `api/config.php` and update your DB credentials
5. Open `http://localhost/Taskflow-project` in your browser
